# Building LLM Applications

This workspace contains several example assistants built using Python:

- Language Translator Assistant
- Question Generator Assistant
- Study Assistant
- Tone Modifier Assistant

Each assistant includes an `app.py` file that can be run individually.

## Usage

Navigate to the desired directory and run:

```bash
python app.py
```

## Structure

The workspace is organized with each assistant in its own folder. Feel free to explore and modify the code to build your own AI-powered applications.
